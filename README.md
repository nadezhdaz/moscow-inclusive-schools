# Школы Москвы для детей с РАС

Практический родительский навигатор по государственным и частным школам, специальным площадкам, ресурсным классам и АООП 8.1–8.4 в Москве, Химках и Люберцах.

Отчёт включает:

- 105 релевантных вариантов обучения в едином каталоге, включая все организации из официального навигатора ГППЦ, где опубликованы программы 8.x;
- 28 частных вариантов — только с прямым или честно помеченным косвенным свидетельством работы с РАС, ОВЗ, ЗПР, СДВГ или другими особенностями;
- 58 адресных строк из навигатора ГППЦ и школьное отделение ФРЦ МГППУ;
- сведения из официальных источников и тематических Telegram-групп;
- фильтры по округу, программе, типу организации, подтверждённости и ступени обучения.

Сайт: <https://nadezhdaz.github.io/moscow-inclusive-schools/>

Источники проверены 14 июля 2026 года. В версии 5.5 уточнён принцип сохранения сведений: в основной каталог попадает любая релевантная информация — действующий класс, прошлогодний набор, единичное родительское сообщение или инициатива по сбору класса — с честной пометкой степени подтверждения. Дополнены №830, Курчатовская школа, Куркино, Отрада, семейный класс в Тушине и инициатива в Крылатском. Навигатор не гарантирует зачисление и не заменяет заключение ЦПМПК или личный визит в школу.
