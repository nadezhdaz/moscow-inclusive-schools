# Школы Москвы для детей с РАС

Практический родительский навигатор по государственным и частным школам, специальным площадкам, ресурсным классам и АООП 8.1–8.4 в Москве, Химках и Люберцах.

Отчёт включает:

- 71 релевантную школу, класс или проект в едином каталоге;
- 27 частных вариантов — только с прямым или честно помеченным косвенным свидетельством работы с РАС, ОВЗ, ЗПР, СДВГ или другими особенностями;
- 59 адресных строк официальных специальных площадок;
- сведения из официальных источников и тематических Telegram-групп;
- фильтры по округу, программе, типу организации, подтверждённости и ступени обучения.
